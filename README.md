# Alura Store: Análisis de Ventas y Optimización de Negocio 

Este repositorio contiene el desarrollo del desafío del programa de formación en **Data Science de Alura LATAM (Oracle Next Education)**. El objetivo principal es asesorar al Sr. Juan sobre la viabilidad de sus unidades de negocio, analizando el rendimiento de cuatro tiendas distintas para recomendar cuál debería ser vendida.

##  Descripción del Proyecto

El desafío pone en práctica habilidades de análisis exploratorio de datos (EDA), visualización y generación de reportes utilizando herramientas de ciencia de datos para resolver un problema de toma de decisiones empresariales.

##  Objetivo del Análisis

Realizar un análisis detallado de los datos de ventas para:
- **Ingresos:** Calcular los ingresos totales por tienda.
- **Inventario:** Identificar productos y categorías más y menos vendidos.
- **Satisfacción:** Calcular calificaciones promedio de los clientes por tienda.
- **Logística:** Evaluar el costo de envío promedio por tienda.
- **Finanzas:** Analizar los métodos de pago utilizados.

##  Herramientas Utilizadas

- **Python 3.x**
- **Pandas** (Procesamiento y limpieza de datos)
- **Matplotlib** (Visualización de datos)



##  Conclusiones y Recomendación Final

Tras evaluar todas las métricas clave, se recomendó **vender la Tienda 4**, debido a los siguientes hallazgos:

1. **Ingresos Críticos:** Presenta los ingresos totales más bajos de la cadena.
2. **Baja Satisfacción:** Las calificaciones de los clientes son deficientes en comparación con las otras tiendas.
3. **Escaso Volumen:** Menor cantidad y diversidad de productos vendidos (falta de productos "estrella").
4. **Adopción de Pagos:** Bajo uso y variedad de métodos de pago por parte de los clientes.
